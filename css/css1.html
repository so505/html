css 기초

css 적용순서

  1. 웹&앱디자인을 완료한다.
  2. 위 디자인에 맞게 HTML을 모두 작성완료한다.
  3. HTML을 꾸미기위한 CSS를 작성한다
  1. 디자인하려는 대상의 태그 or 클래스 or 아이디를 확인한다.
  2. 위 확인대상을 css 선택자로 작성한다.
  3. 위 선택자에서 적용하고자 하는 디자인 속성을 작성한다
  4. 속성에 맞는 값을 적용한다

CSS 작성방법

  • head태그 내에 style태그를 산입해 css를 작성한다.
  • 별도의 css파일을 생성해서 head태그에 link로 연결해서 작성한다.
  • 디자인 적용대상 태그에 직접css를 작성한다.
