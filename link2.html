상대경로

절대적 웹 주소가 아닌 현재 작업중인 파일을 기준으로 해당 폴더 같은 또는 폴더 또는 하위 폴더 또는 상위 폴더로 원하는 파일의 위치를 상대적으로 지정하는 경로

절대경로 웹주소 작성시 시작->http

상대경로 작성 시 시작 -> ./

상대경로 작성순서

1. 현재 작업중인 파일의 위치를 확인한다.

2. 위 1 기준 연결하고 싶은 파일의 위치를 확인한다.

3. ./로 경로를 시작하며 원하는 상대경로를 완성한다.
